Durant l'année nous allons beaucoup utiliser les outils Google afin de collaborer. Pour se faire nous devons utiliser un compte Google. Si vous possédez déjà un compte vous pouvez sauter cette étape.

Accéder à la page de création de compte

Ouvrer votre navigateur web préféré (Google Chrome, Mozilla Firefox, etc.).

Allez sur la page suivante : https://accounts.google.com/signup

Suiver le processus de création de compte

En ce qui concerne vos informations personnelles (nom, prénom, date de naissance etc) vous pouvez mettre des informations "bidons" si vous le souhaitez. Néanmoins vous devrez mettre votre numéro de téléphone afin de valider votre compte.

Une fois connecté à votre compte Google, aller sur Pronote : Outils pédagogiques > Ressources pédagogiques > Ressources pour les élèves

Cliquer sur le lien "Comptes-Google-1TSD-annee-2023" puis ajouter votre adresse mail Google en face de votre nom.

Ouvrer un nouvel onglet et taper "google drive", cliquez sur le premier lien :

Ensuite cliquez sur "Accéder à Drive"

Cliquer sur "Nouveau" en haut à gauche, puis "Google Docs"

Renommer votre document "TP compte rendu"

Déplacer votre fichier dans le dossier "1TSD/Communication"

Écrire un court paragraphe faisant le compte rendu de cette séance de TP.

Pour partager votre document avec votre voisin ou voisine, cliquer sur "Partager"

Consulter le fichier contenant les adresses mails de tous les étudiants de la classe :

Puis partager votre fichier avec vos voisins/voisines.

Modifier les droits afin que votre invité puisse commenter le fichier mais pas le modifier.

Maintenant que vous avez un fichier partagé vous allez pouvoir suggérer des modifications à votre camarade.

Proposer un nouveau point concernant l'ordre du jour :

Ajouter un commentaire sur le ‘g' du point numéro 2

Selectionner tout le texte et changer la police en "Roboto"

Suggérer également un titre à ce document :

Attendre que votre camarade ait fini de faire toutes ces suggestions sur votre document puis passer à l'étape suivante.

Traiter des suggestions/commentaires

Consulter votre Google Doc modifié. Afficher l'historique des commentaires.

Traiter chaque suggestion en l'acceptant/refusant/metant un commentaire pour plus d'informations

Tchatter avec vos collaborateurs

Vous pouvez discuter avec vos collaborateurs en ligne. Pour cela cliquer sur "Afficher le chat" :

Envoyer un message à vos collaborateurs :

Attendez sa réponse puis discutez entre vous.

Créer une réunion Google Meet

Il est également possible de créer un appel vidéo afin de travailler sur le document tout en discutant.

Cliquer sur l'icon de la caméra

Choisir "Démarrer une nouvelle réunion"

Copier/coller le lien de la réunion et envoyer le à votre camarade :

Attendre que votre collaborateur rejoigne la réunion. Puis cliquer sur "Présenter"

Votre camarade peut désormais visualiser vos actions tout en discutant avec vous. Pour agrandir la vidéo cliquez sur "Déplacer vers un onglet" :
